# 📺 Smart Social Media Comment Moderation System

An AI-powered desktop application that analyzes YouTube comments using Natural Language Processing (NLP), classifies their sentiment, flags potentially harmful comments, visualizes moderation insights, and exports detailed reports to Excel.

---

## 📌 Overview

The **Smart Social Media Comment Moderation System** is designed to simplify the process of moderating large volumes of YouTube comments. Instead of manually reviewing thousands of comments, the application automatically analyzes each comment, identifies its sentiment, flags comments that may require moderator attention, and presents the results through an intuitive dashboard.

The application also generates a downloadable Excel report containing all moderation results for further analysis and record keeping.

---

## ✨ Features

* 🔍 Analyze YouTube comments using NLP-based sentiment analysis
* 😊 Classify comments as **Positive**, **Neutral**, or **Negative**
* 🚩 Automatically flag potentially harmful or inappropriate comments
* 📊 Visualize moderation insights using charts and graphs
* ☁️ Generate a word cloud from frequently used words
* 📈 Display sentiment and moderation statistics
* 📄 Export analyzed results to an Excel (.xlsx) report
* 🖥️ User-friendly desktop interface built with Python

## ⚙️ How It Works

1. Upload link of youtubr video.
2. The application processes each comment using sentiment analysis.
3. Each comment is classified as Positive, Neutral, or Negative.
4. Harmful or highly negative comments are automatically flagged.
5. Interactive charts summarize the moderation results.
6. A detailed Excel report is generated for download.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/manshavEI/Smart-social-media-comment-moderation-system.git
```

Navigate to the project directory:

```bash
cd Smart-social-media-comment-moderation-system
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
